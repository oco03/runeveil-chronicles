# ✦ Runeveil Chronicles ✦

Un RPG de criaturas en pixel art estilo Pokémon, creado con React.

![Runeveil](https://img.shields.io/badge/Runeveil-Chronicles-9b59b6?style=for-the-badge)

## 🌍 Lore

En el mundo de **Runeveil**, cinco cristales ancestrales mantienen el equilibrio entre las fuerzas elementales. Hace mil años, el **Sabio Oscuro Malachar** intentó fusionar los cristales para obtener poder absoluto, pero fue derrotado por los **Guardianes Rúnicos** y sellado en una prisión dimensional.

Ahora, las runas se debilitan. Grietas dimensionales se abren y criaturas llamadas **Veilborn** emergen del Velo entre mundos. Tú eres el último descendiente de los Guardianes: captura y entrena Veilborn para restaurar los cristales antes de que Malachar regrese.

## 🎮 Controles

| Acción | Teclado | Móvil |
|--------|---------|-------|
| Mover | WASD / Flechas | D-Pad en pantalla |
| Ver equipo | P | Botón central D-Pad |
| Interactuar | Enter / Espacio | Tocar botón |
